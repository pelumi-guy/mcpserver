# Security Policy

## Supported Versions

This project is currently maintained on the main branch.
Security fixes are applied to the latest codebase.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately.

Preferred contact:
- GitHub profile: https://github.com/Ekundayo-tech

Please include:
- A clear description of the issue
- Impact assessment
- Reproduction steps or proof of concept
- Suggested mitigation, if available

## Response Expectations

- Initial acknowledgment target: within 5 business days
- Triage and severity assessment: as quickly as possible
- Fix timeline: based on severity and complexity

## Disclosure

Please avoid public disclosure until a fix is available and released.
Coordinated disclosure is appreciated.
